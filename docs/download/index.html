Source

For installation instructions head to the documentation page.

MacPorts

Stable versions are available on Mac OS via Macports as follows:

sudo port install giza

Development version

You can browse the source code online through the subversion interface.
Check out a development copy using the following command:

svn co http://svn.code.sf.net/p/giza/code/ giza

If you would like to contribute code, please send an email to the address on the contact page.

Stable versions

Stable releases are available from the sourceforge page.
